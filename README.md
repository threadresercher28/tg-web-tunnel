# tg-web-tunnel
Проект превращает сайт в проски для Telegram Desktop (Для Компьютера) программма разворачивает локальный прокси на порту и IP указаном в ```config.py```:
```python3
LISTEN_PORT = 8021 # Порт локального прокси
PHP_PROXY_SERVERS = ["http://internet.ru/prx.php"] # Список серверов
CHUNK_SIZE = 8192 # Количество чанков на запрос.
MAX_RETRIES = 2 #Максимальное каличество пинг запросов.
RETRY_STATUS_CODES = [500, 502, 503, 504] #коды при которыйх запрос отправляется повторно
LOG_LEVEL = "INFO" # Уровень логирования.
backoff_factor = 0.5  # Задержка перед повторным запросом в случае  ошибки.
```
После подключитесь в настройках клиента ```Настройки -> Продвинутые настройки -> Тип ссоединения -> Добавить прокси``` там выбераем ```HTTP```, в хост: ```0.0.0.0```,```127.0.0.1```,```localhost``` любой вариант, порт из ```LISTEN_PORT = [Порт]```.
