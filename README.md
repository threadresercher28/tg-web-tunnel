# tg-web-tunnel
Проект превращает сайт в проски для Telegram Desktop (Для Компьютера) программма разворачивает локальный прокси на порту и IP указаном в ```config.py```:
```python3
LISTEN_PORT = 8021 # Порт локального прокси
PHP_PROXY_SERVERS = ["http://internet.ru/prx.php"] # Список серверов
CHUNK_SIZE = 8192 # Количество чанков на запрос.
MAX_RETRIES = 2 #Максимальное каличество пинг пакетов.
RETRY_STATUS_CODES = [500, 502, 503, 504] #коды при которыйх запрос отправляется повторно
```


## Как работает ?
1. Заливаем  prx.php на сайт с поддержкой ```php-curl```
